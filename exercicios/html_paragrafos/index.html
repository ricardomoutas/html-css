Explorando Parágrafos e Quebras em HTML5

Em HTML, um parágrafo é criado com a tag <p>. O navegador trata todo o texto dentro dessa tag como parte de um mesmo parágrafo, independentemente dos “enters” ou espaços adicionados no código.

Para quebrar uma linha
Num local específico, utiliza-se a tag <br>. Observa que a frase acima foi dividida em duas linhas no navegador usando este recurso.

Exemplos de caracteres especiais em HTML:
Sinal de menor que: <
Sinal de maior que: >
